# Statistical tests

## Wald test
- Used to test whether or not an explanatory variable in a regression (i.e. a $\beta$) is significant, meaning it contributes predictive power to the model. The null hypothesis asserts that $\beta = 0$, i.e. that the variable has no explanatory value.
