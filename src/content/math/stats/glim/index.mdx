# Generalized Linear Models (GLIM)

- Not general linear models (GLM).
- Every GLIM has three components
    - Random component
        - The distribution of the response variable
            - In classic linear regression, the _normal distribution_
            - In binary logistic regression, the _binomial distribution_
    - Systematic component
        - Specifies the explanatory variables
    - Link function $\eta$ or $g(\mu)$
        - Specifies how the expecetd value of the response relates to the linear combination of the explanatory variables.
- Parameter estimation uses **maximum likelihood estimation (MLE)** rather than ordinary least squares.
