## Overview
- pUC19 forms **backbone**: **origin of replication** and **drug resistance gene**
- [mUAV](https://www.addgene.org/102680/) contains the sequence for amilCP, which we can amplify using PCR, and then assembly together using Gibson assembly
- use 15 to 40bp overlaps with $T_m > 48\degC$
