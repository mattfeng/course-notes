## Gatsby

### Sass
```bash
npm install sass gatsby-plugin-sass
```

### [Markdown Extended (MDX)](https://www.gatsbyjs.com/plugins/gatsby-plugin-mdx/)
```bash
npm install gatsby-plugin-mdx @mdx-js/mdx @mdx-js/react
```

```js
// gatsby-config.js
module.exports = {
  plugins: [
    {
      resolve: `gatsby-source-filesystem`,
      options: {
        name: `pages`,
        path: `${__dirname}/src/pages/`,
      },
    },
    {
      resolve: `gatsby-plugin-mdx`,
      options: {
        defaultLayouts: {},
        gatsbyRemarkPlugins: [],
        remarkPlugins: [],
        rehypePlugins: [],
      }
    }
  ],
}
```


### [Katex](https://katex.org/docs/supported.html)

```bash
npm install remark-math@3 rehype-katex@3 katex
```

- create a `macros` option in `rehype-katex` to use **gdef**

- how to hack in mhchem
```js
// rehype-katex/index.js
let katex = require('katex')
require("katex/dist/contrib/mhchem.js")
katex = katex.renderToString
```

### Github pages

```bash
npm install gh-pages --save-dev
```

```json
# package.json
{
  "scripts": {
    "deploy": "gatsby build --prefix-paths && gh-pages -d public",
  }
}
```

```js
// gatsby-config.js
module.exports = {
    pathPrefix: '/reading-group'
}
```
